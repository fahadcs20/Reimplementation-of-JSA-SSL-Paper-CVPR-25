# Reimplementation-of-JSA-SSL-Paper-CVPR-25
A PyTorch reimplementation of "Improving Sound Source Localization with Joint Slot Attention on Image and Audio" using a dummy dataset. Features Joint Slot Attention, cross-modal matching, and full loss components for disentangling audio-visual representations.
